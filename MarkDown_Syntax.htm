Markdown Cheatsheet

Hello! This is markdown cheat sheet for markdown beginner users. I hope you find this helpful!

こんにちは！　Markdownのカンニングペーパーです。ご使用ください。

1. Headings・ヘディング

To create a heading, add sign「#」 in front of a word or phrase.
The number of the「#」signs you use corresponds to the heading level. For example, to if you want to create a heading level three (<h3>), add three signs (e.g., ### My Header).

ヘディングを作成したい場合、テキストの前に「＃」を追加してください。
「＃」の数によって、ヘディングレベルが変更されます。例えば、ヘディングレベル３（<h3>）を作成したい場合、「＃」を3回入力してください（例：###私のヘッダー）。


Markdown Rendered Output Try it!
# Heading level 1

Heading level 1
## Heading level 2

Heading Level 2
### Heading level 3

Heading level 3
#### Heading level 4

Heading level 4

2. Bold・太字

To bold text, add two asterisks or underscores before and after a word or phrase.
To bold the middle of a word for emphasis, add two asterisks without spaces around the letters.

テキストを太字にしたい場合、テキストの前と後ろに「**」や「__」追加してください。　
言葉の真ん中の文字を太字にしたい場合、スペースをせず、文字の前後ろに「**」を追加してください（例：bea**uti**ful）。


Markdown Rendered Output Try it!
I love **bold text** I love bold text
I love __bold text__ I love bold text
Love**is**bold Loveisbold

3. Italic・斜体

To italicize text, add one asterisk or underscore before and after a word or phrase.
To italicize the middle of a word for emphasis, add one asterisk without spaces around the letters.

テキストを斜体にしたい場合、テキストの前と後ろにアスタリスクやアンダースコアを１つ追加してください。
言葉の各文字を斜体にしたい場合、その文字の前と後ろにアスタリスクを１つ追加してください。


Markdown Rendered Output Try it!
I love *italic font* I love italic font
I love _italic font_ I love italic font
Ilove*italicfont* Iloveitalicfont

4. Bold & Italic・太字と斜体

To emphasize text with bold and italics at the same time, add three asterisks or underscores before and after a word or phrase.

テキストを太字と斜体にしたい場合、テキストの前と後ろにアスタリスやアンダースコアを３つを追加してください。

Markdown Rendered Output Try it!
This text is ***really important*** This text is really important
This text is ___really important___ This text is really important
This text is __*really important*__ This text is really important
This text is **_really important_** This text is really important

6. Ordered Lists・順序付きリスト

To create an ordered list, add line items with numbers followed by periods.
The numbers don’t have to be in numerical order, but the list should start with the number one.

順序付きリストを作成したい場合、行ごとに番号が付いているテキストを入力してください。
最初の行のテキストに必ず「１」番号を追加してください。その後の番号が後件ではなくても、順序付きリストとして表示されます。


Markdown Rendered Output Try it!
1. First item
2. Second item
3. Third item
4. Forth item
  1. First item
  2. Second item
  3. Third item
  4. Forth item
1. First item
1. Second item
1. Third item
1. Forth item
  1. First item
  2. Second item
  3. Third item
  4. Forth item
1. First item
8. Second item
11. Third item
3. Forth item

  2. Second item

  4. Forth item
